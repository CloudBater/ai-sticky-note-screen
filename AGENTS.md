# Working Agreement

This project was built with AI assistance, but I made the final scope decisions.

## How I used AI

- Drafting a small fullstack structure.
- Checking the scope against the grading rubric.
- Writing first-pass tests for small helper functions.
- Reviewing `PLAN.md` wording so the refusals stayed clear.

## How I kept ownership

- I read the topic, rubric, and examples before implementation.
- I chose a conservative scope: daily FX reference data, not trading or predictions.
- I kept the implementation small enough to explain in an interview.
- I reviewed diffs before committing.

## Project rules

- The frontend calls only my backend.
- The backend calls Frankfurter.
- UI copy must not say "real-time", "AI prediction", or "trade".
- Unsupported currencies should be handled explicitly.
- No secrets are needed for this topic, and none should be added.
